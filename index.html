CSS custom properties

CSS custom properties are like CSS variables. They help reduce repetition of code and make it easier to update the code. Instead of needing to change the code in several places, the code only needs to be changed in one place and the styles update automatically.

Learn more in depth about CSS custom properties at CSS Tricks.

Naming convention and use

Custom properties have a specific naming convention and use. Each custom property must start with two dashes (--) and be within a selector.

See the Pen CSS Custom Properties - Naming and use by Zach Kelley (@zkelley23) on CodePen.

Properties are accessed by putting the property inside var(). Below, --blue and --black are the custom properties with #67CBF0 and #222 as their respective values. var(--blue) and var(--black) are the variables.

Color

Using custom properties to set color themes is very useful. By setting the colors in the html{} css, the colors can be accessed throughout the stylesheet. If the colors ever need to be changed, the only need to be updated once.

Spacing

Using custom properties to set spacing (padding and margin) is also very useful. Again, by setting the spacing in the html{} css, the spacing can be accessed throughout the stylesheet. If the spacing needs to be adjusted at all, it only needs to be changed in one place.
